# PRD: 関係法規マスター（柔道整復師国試学習アプリ）

## 概要

柔道整復師国家試験の「関係法規」科目（過去34年分393問）を、学生が100%正解できるようにする学習アプリ。

## 背景・課題

- 関係法規は暗記科目だが、類似概念の混同や忘却により得点が安定しない
- 紙ベースの問題集では「間違えた問題の繰り返し」が非効率
- 学生個別の弱点を把握し、最適な復習タイミングを提供する仕組みがない

## ターゲットユーザー

| 属性 | 詳細 |
|------|------|
| 対象 | 柔道整復学科 1年生（前期） |
| 人数 | 約50人（午前コース30人 + 午後コース20人） |
| 年齢層 | 18-20代前半が中心 |
| 利用端末 | スマートフォン・タブレット |

## 成功指標

| KPI | 目標値 |
|-----|--------|
| DAU/MAU比率 | 40%以上 |
| 2日目継続率 | 70%以上 |
| 7日目継続率 | 50%以上 |
| 平均セッション長 | 5-8分 |
| 練習テスト合格率(80%) | 全学生の80%以上 |

## 最終ゴール

終講試験および国家試験の関係法規セクションの得点向上。
